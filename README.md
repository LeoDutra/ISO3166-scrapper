ISO3361 scrapper
================
*Scraps ISO3166-1 Alpha 2 and ISO3166-2 data to JSON*

Running
-------
+ install git
+ run `git clone https://github.com/LeoDutra/ISO3166-scrapper.git`
+ install Node.js
+ run `node app.js`
+ check `ISO3166.json`

current source:
http://www.unece.org/cefact/locode/subdivisions.html
