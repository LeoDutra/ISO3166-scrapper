ISO3361 scrapper
================
*Scraps ISO3166-1 Alpha 2 and ISO3166-2 data to JSON*

Running
-------
  1. install git
  2. run `git clone https://github.com/LeoDutra/ISO3166-scrapper.git`
  3. install Node.js
  4. run `node app.js`
  5. check `ISO3166.json`

current source:
http://www.unece.org/cefact/locode/subdivisions.html
